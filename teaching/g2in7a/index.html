Interprétation

On écrit le programme dans un fichier .ml et on le charge avec l’interpréteur OCaml (dans lequel on peut écrire et tester les fonctions librement).

ocaml
#use "fichier.ml";;

Pour quitter :

exit 0;;

Compilation

On écrit le programme dans un fichier .ml, on le compile pour obtenir un exécutable.

ocamlc fichier.ml -o fichier

TP1

Exercice 1

  1. fact n qui calcule la factorielle !n d’un entier
  2. pow n k qui calcule la puissance n^k (pour deux entiers)
  3. fact_acc n acc qui calcule la factorielle en stockant en en mettant à jour le résultat à chaque étape dans son second argument
  4. append l1 l2 qui calcule la concaténation de deux listes (par exemple append [1; 2] [3; 4] = [1; 2; 3; 4])
  5. rev l qui calcule le renversement d’une liste (par exemple rev [1; 2; 3; 4] = [4; 3; 2; 1])
  6. rev_acc l acc qui calcule le renversement d’une liste en transférant les éléments de l dans acc puis en renvoyant acc

Exercice 2

On suppose qu’on travaille sur des listes sans répétitions (ensembles mathématiques).

  1. appartient x l qui détermine si l’élément e est dans la liste n
  2. inter l1 l2 qui calcule l’intersection de deux listes sans répétitions (par exemple inter [1; 2] [2; 4] = [2])
  3. union l1 l2 qui calcule l’union de deux listes sans répétitions (par exemple union [1; 2] [2; 4] = [1; 2; 4])

On suppose maintenant que l’on travaille sur des listes sans répétions mais aussi triées. On veut définir des fonctions de façon intelligente en prenant en compte le fait qu’elles sont triées.

Bonus

  1. diff l1 l2 qui calcule les éléments de l1 sans ceux de l2 (par exemple diff [1; 2] [2; 4] = [1])
  2. subset l1 l2 qui calcule si l1 est un sous ensemble de l2 (tous deux supposés sans répétitions)
  3. remove1 x l calcule la liste obtenue si on retire le premier x dans l
  4. remove_all x l calcule la liste obtenue en retirant tous les x dans l
  5. zip l1 l2 qui fusionne deux listes de même taille en mettant chaque élément par paire (par exemple zip [1; 2; 3] [4; 5; 6] = [(1, 4); (2, 5); (3, 6)])
  6. unzip l qui prend une liste de paires et effectue l’opération inverse (par exemple unzip [(1, 4); (2, 5); (3, 6)] = ([1; 2; 3]; [4; 5; 6]))
